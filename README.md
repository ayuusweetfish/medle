# Medle

[**medle.0-th.art**](https://medle.0-th.art/)

旋律猜谜游戏，每日更新~  
Daily melody puzzle game (available in en/zh)

## [zh] 服务端程序

安装 [Deno](https://deno.land/) (1.21.0)，然后执行命令

```sh
deno run --allow-net --allow-read --allow-env server.js
```
